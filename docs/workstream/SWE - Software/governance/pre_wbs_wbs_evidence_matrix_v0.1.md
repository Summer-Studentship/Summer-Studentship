## Overall disposition

The pre-WBS repository contains useful history and prototype scaffolding for G0, especially around CMake, vcpkg, presets and Qt experimentation. It also contains material divergences from the accepted stack and architecture. Each assessed Deliverable therefore remains open until its formal Work Package produces and passes the stated acceptance evidence.
