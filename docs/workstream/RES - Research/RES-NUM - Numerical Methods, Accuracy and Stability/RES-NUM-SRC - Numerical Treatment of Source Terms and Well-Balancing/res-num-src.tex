\subsection{Selected Approach and Rationale}
\label{sec:res-num-src-selected-approach}

% TODO: Record the selected approach only after sufficient evidence exists.
% State the alternatives rejected and the reasons for rejection.

The selected regional source-term approach is:

\begin{itemize}
  \item hydrostatic reconstruction for bed-slope balance;
  \item matching second-order bed-gradient source in cells;
  \item dynamic bed updates at Runge--Kutta stage times;
  \item Strang-split Manning friction;
  \item polynomial sponge ramp with baseline $p=2$,
    $A_{\mathrm{target}}=0.01$ and $L_{\mathrm{sp}}=50$--$100~\mathrm{km}$.
\end{itemize}
