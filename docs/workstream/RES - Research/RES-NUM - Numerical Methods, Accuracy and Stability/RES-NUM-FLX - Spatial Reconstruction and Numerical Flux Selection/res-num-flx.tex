\subsection{Discrete Formulation}
\label{sec:res-num-flx-discrete-formulation}

The regional reconstruction state is
$\mathbf{W}=[\eta,u,v,b]^{\mathsf{T}}$. The selected reconstruction is
MUSCL piecewise-linear reconstruction using weighted least-squares
gradients and a Barth--Jespersen limiter, as defined in
\cref{eq:res-num-spec-regional-reconstruction-state,eq:res-num-spec-regional-muscl-reconstruction,eq:res-num-spec-regional-ls-gradient}.
Reconstruction reverts to first order near wet/dry fronts where needed
for positivity.

is the fallback near dry, degenerate or unsafe states. Face states are
rotated into normal/tangential coordinates before the Riemann solve, as

\subsection{Selected Approach and Rationale}
\label{sec:res-num-flx-selected-approach}

% TODO: Record the selected approach only after sufficient evidence exists.
% State the alternatives rejected and the reasons for rejection.

Selected regional flux/reconstruction hierarchy:

\begin{itemize}
  \item second-order MUSCL reconstruction in smooth wet regions;
  \item weighted least-squares gradients on polygonal meshes;
  \item Barth--Jespersen multidimensional limiting;
  \item hydrostatically reconstructed face states before fluxing;
  \item HLLC for the normal shallow-water Riemann problem in valid wet
    states;
  \item HLL/HLLE fallback for dry, nearly dry or otherwise unsafe
    states.
\end{itemize}
