\subsection{Continuous Governing Model}
\label{sec:res-mod-lrg-continuous-governing-model}

The selected large-domain regional representation is the horizontal
two-dimensional depth-averaged NLSWE model defined in
`RES-MOD-EQS` and consolidated in `RES-MOD-SPEC`. It represents
Tōhoku-scale generation, propagation, nearshore transformation and
inundation without applying local three-dimensional CFD resolution
across the whole domain.

\subsection{Selected Approach and Rationale}
\label{sec:res-mod-lrg-selected-approach}

% TODO: Record the selected approach only after sufficient evidence exists.
% State the alternatives rejected and the reasons for rejection.

Use the regional 2D NLSWE finite-volume solver as the large-domain
model. Preserve local three-dimensional modelling only where
structure-scale pressure, overtopping, local air--water interface
topology or FSI requires vertically resolved flow.

artificial-boundary placement, sponge-layer placement, local 3D
